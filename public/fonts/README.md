# Custom Fonts

This directory contains custom typefaces used in the Society Protocol design.

## PP Telegraf

The `@font-face` declaration in `src/styles/globals.css` expects:

- `PPTelegraf-Regular.woff2`
- `PPTelegraf-Regular.woff`

These files should be sourced from [pangrampangram.com](https://pangrampangram.com/fonts/pp-telegraf) and placed in this directory.

**Note:** PP Telegraf is referenced in the Figma design and used for display text (`.font-display` class in Tailwind).
